# <u> ---------------- Networking-Fundamentals --------------- <u/>

A comprehensive collection of Networking fundamental concepts.

## <u> Topics Covered <u/>

  - OSI Model — all 7 layers with real-world examples
  - TCP/IP, UDP — differences, use cases, 3-way handshake
  - DNS — how resolution works (A, CNAME, MX, TXT records)
  - HTTP/HTTPS — request/response, status codes, headers, TLS, SSH
  - IP addressing — IPv4, IPv6, subnetting, CIDR notation (/24, /16)
  - NAT, Firewalls, Load Balancers — conceptual understanding
  - VPN basics and tunneling concepts
  - Common ports: 22(SSH), 80(HTTP), 443(HTTPS), 3306(MySQL), 5432(Postgres)
  - Packet capture basics with tcpdump/Wireshark
